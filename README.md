# ShadowGuess
A game that allows people to shadow puppet a story that other people have to guess the name to.

![Logo](./image1.png)

**The entire demo of the project can be found on [YouTube](https://youtu.be/VMIgoVEDtUk).**

## Technologies and Tools
* Python 
* MediaPipe
* PyGame
* OpenCV
